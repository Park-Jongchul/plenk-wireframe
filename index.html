1. 프로젝트 개요

프로젝트 목적

사용자가 여행에 필요한 정보를 입력하면 여행 일정, 관광지, 맛집, 숙소, 이동 동선을 사용자의 취향에 맞게 추천하는 한국 여행 일정 추천 웹 서비스이다.

초기 개발 범위는 웹 브라우저 기준으로 하며, PC와 모바일 반응형 화면을 모두 고려한다.

주요 기능

여행 정보 입력일정 생성장소 추천일정 수정저장공유

컴포넌트/이벤트 명세

① 여행 지역 Dropdown
필수. 클릭 시 지역 선택 모달 오픈. 인기/전체 지역 검색 가능. 선택값은 trip.regionId에 저장.
② 여행 기간 Date Range Picker
필수. 시작일/종료일 선택. 종료일이 시작일보다 빠르면 ERR-002 표시. 최대 30일까지 허용.
③ 여행 인원 Selector
필수. 성인/아동/반려동물 여부 선택. 최소 1명, 최대 20명.
④ 이동 수단 Radio Card
필수. 대중교통/자동차/도보/택시 중 1개 선택. 결과 화면 동선 계산 기준으로 사용.
⑤ 다음 버튼
클릭 시 WEB-002 필수값 Validation 실행. 성공 시 입력값 저장 후 WEB-003 이동. 실패 시 첫 번째 오류 영역으로 스크롤.
